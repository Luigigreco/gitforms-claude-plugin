# GitForms Assistant Skill

AI-powered assistant for managing contact forms with natural language.

## Auto-Activation Triggers

This skill activates when user mentions:
- "contact form", "lead capture", "form submissions"
- "gitforms", "github issues form"
- "manage contacts", "track leads"
- "form backend", "serverless forms"

## Capabilities

### Natural Language Commands

Instead of slash commands, users can say:

**Form Creation:**
- "Create a contact form with name, email, and message"
- "I need a lead capture form for my landing page"
- "Set up a form that saves to GitHub Issues"

**Contact Management:**
- "Show me all leads from this week"
- "List contacts with budget over $10k"
- "Export all enterprise leads to CSV"

**Analysis:**
- "Analyze my form submissions"
- "Which leads should I follow up first?"
- "Show trends for the last month"
