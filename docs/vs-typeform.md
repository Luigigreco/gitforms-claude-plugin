# GitForms vs Typeform - Comparison

> Why choose GitForms over Typeform for your forms?

## Quick Comparison

| Feature | GitForms | Typeform |
|---------|----------|----------|
| **Price** | 100% Free | From $25/mo |
| **Privacy** | Zero tracking | Heavy analytics |
| **Open Source** | CC BY-NC-SA 4.0 | Closed source |
| **Self-Hosted** | Your GitHub | Their servers |
| **Branding** | No branding | Branding on free |
| **MCP Server** | Official | None |
| **API Access** | Always free | Paid plans only |

## Pricing Breakdown

### GitForms
- **Free**: Unlimited forms, unlimited submissions
- **No hidden costs**: Ever

### Typeform
- **Free**: 10 responses/month, Typeform branding
- **Basic**: $25/mo - 100 responses
- **Plus**: $50/mo - 1,000 responses
- **Business**: $83/mo - 10,000 responses

## Privacy Comparison

**GitForms**:
- No tracking pixels
- No analytics cookies
- Data stored in YOUR repository
- GDPR compliant by design

**Typeform**:
- Multiple tracking scripts
- Third-party integrations
- Data on their servers
- Requires GDPR configuration

## Developer Experience

**GitForms**:
- Claude Code integration via MCP
- Natural language commands
- Full API access (free)
- Open source customization

**Typeform**:
- Visual form builder
- Limited API (paid)
- No AI assistant integration
- No self-hosting option

## Conclusion

**Choose GitForms** if you want:
- Zero cost
- Full privacy
- Developer-friendly tools
- AI assistant integration

**Choose Typeform** if you need:
- Visual form design
- Non-technical team
- Enterprise support
- Complex form logic
